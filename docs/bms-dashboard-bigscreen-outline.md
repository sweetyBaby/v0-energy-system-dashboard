# BMS 数据监测云平台大屏梳理

## 1. 推荐的大屏一级模块

1. 实时监测
   展示项目概况、实时功率、SOC、SOH、总压总流、在线设备数、活动告警数、通讯质量，以及充放电功率曲线、能量曲线。
2. 综合效能
   重点展示功率、系统效率、可用容量、等效循环、累计充放电量、累计充放电容量，并用组合图展示容量/能量/效率的周期趋势。
3. 分析诊断
   展示电芯压差分析、温差分析、最大最小电压分析、异常簇定位、阈值越限趋势。
4. 历史查询
   展示 BCU 运行状态查询、SOC/SOH 历史轨迹、告警日志、事件追溯、设备状态切换记录。
5. 报表中心
   展示日报、月报、年报、专题报表的生成状态、导出状态和字段覆盖情况。

## 2. 综合效能建议指标

- 实时功率：当前充电功率 / 放电功率
- 系统效率：周期平均系统效率
- 可用容量：当前可用容量 / 额定容量
- 等效循环：按日、月、累计展示
- 充放电量：日、月、年、累计
- 充放电容量：Ah 统计
- 容量效率：放电容量 / 充电容量
- 能量效率：放电电量 / 充电电量

## 3. 报表建议结构

- 日报：运行概况、充放电统计、功率曲线、SOC/SOH、告警摘要
- 月报：能效统计、等效循环、容量衰减、可用率、告警闭环率
- 年报：累计电量、累计收益、设备健康度、关键异常事件
- 专题报表：压差分析、温差分析、告警专题、设备可用率专题

## 4. 当前页面已对应的实现

- 实时监测：项目概况、实时状态、功率曲线、能量曲线、充放电统计
- 综合效能：新增综合效能总览面板，支持截图风格的组合图
- 分析诊断：压差分析、温差分析、最大最小电压分析
- 历史查询：BCU 状态查询、告警日志
- 报表中心：新增报表概览与建议字段
